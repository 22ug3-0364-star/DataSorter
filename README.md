# DataSorter

A Java project that compares different sorting algorithms.

## Algorithms Included
- Bubble Sort
- Merge Sort
- Quick Sort

## Main File
`src/MainProgram.java`

## How to Run
1. Open the project in VS Code or IntelliJ IDEA.
2. Compile and run the `MainProgram.java` file.
3. Observe sorting performance outputs.
